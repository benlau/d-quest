Detailed Description

The meta info of a database model.

Each of the derived class of DQModel must be associated with a DQModelMetaInfo. Otherwise it is not usable.

However, user do not need to generate by themself. User just need to use DQ_MODEL/DQ_DECLARE_MODEL macro pair to declare the database field of a DQModel, it will generate the class automatically.

When it is created , it will set its parent to QCoreApplication, so that it will be destroyed automatically.
