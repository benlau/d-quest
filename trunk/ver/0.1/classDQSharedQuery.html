Detailed Description

DQSharedQuery is the base class of DQQuery that support implicitly data sharing.

DQSharedQuery is the base class of DQQuery that hold all the information for a query. It is a implicitly shared class which is designed for query data exchange between different objects.

Although most of the function return DQSharedQuery instead of DQQuery , user should always use DQQuery. They are exchangable, that means DQSharedQuery can be converted to DQQuery, and vice visa.

Remarks:
It is a implicitly shared class
